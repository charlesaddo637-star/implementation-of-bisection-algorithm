# implementation-of-bisection-algorithm
A Python script to approximate the square root of a number using the **bisection method**.   This project demonstrates numerical methods, convergence checks, and error handling in Python.
## 🚀 Features
- Approximates square roots using the bisection method.
- Handles special cases for `0` and `1`.
- Raises an error for negative inputs.
- Configurable tolerance and maximum iterations.
- Provides clear feedback on convergence or failure.
